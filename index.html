Some Questions

How does javascript works?

Javascript runs with help of javascript engine like Chrome V8 Engine and Spider Monkey. javascript is an interpreted language. That means its execute line by line. First javascript engine parse the javascript code and check the code line by line. if got any error then its shows error or if there is no error then parser create a abstract syntax tree. After that abstract syntax tree converted into machine code. In this step computer understand the machine code and give us the output. Then we can see the result of the javascript code.

What is the difference between local storage and session storage?

Basically session and local storage can store data. But there is few difference between them. Session storage store data for a particular session. that means when a visitor start visiting a web app a session will start. The session storage data exist until the visitor close the web app. After clossing the web app the data will be lost. On the other hand local storage data will not lost if user close the web app. Even user shut down the computer and come back few days later and open the web app the data will not be lost.
